## How to Contribute

To get started with a project:

1. **Fork the repository**
   ```bash
   git clone https://github.com/koushikcs562/gsoc.git
   cd gsoc
Create a branch for your work

git checkout -b your-branch-name


Make your changes

Follow the Contributor Guidance

Add new features or fix issues according to the Ideas List

Commit your work

git add .
git commit -s -m "Description of your work"
Push your branch to GitHub
git push origin your-branch-name
Open a Pull Request

Go to the repository on GitHub

Click Compare & pull request

Provide a clear title and description of your changes
Submitting Your Final Report

Use the Report Template
 to prepare your final report

Include references to your PRs, demos, and results

Submit it according to the GSoC guidelines before the deadline

Best Practices
Keep commits small and focused

Write clear commit messages

Always test your code before creating a PR

Review other contributors’ PRs if possible to learn and provide feedback
